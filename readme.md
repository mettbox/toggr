# togglr

> Toggl time tracking reports CLI

# Install

Install dependencies

```sh
$ npm install
```

Install Package globally

```sh
$ npm install -g .
```

# Usage

```sh
$ togglr COMMAND
running command...

$ togglr (-v|--version|version)
togglr/2.0.0 darwin-x64 node-v15.4.0

$ togglr --help [COMMAND]

USAGE
  $ togglr COMMAND
...
```
