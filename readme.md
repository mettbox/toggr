# toggr

> Toggl time tracking reports CLI

# Install

Install dependencies

```sh
$ npm install
```

Install Package globally

```sh
$ npm install -g .
```

# Usage

```sh
$ toggr COMMAND
running command...

$ toggr (-v|--version|version)
toggr/2.0.0 darwin-x64 node-v15.4.0

$ toggr --help [COMMAND]

USAGE
  $ toggr COMMAND
...
```
